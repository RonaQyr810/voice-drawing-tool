# AI 语音绘图工具

纯语音控制的 Web 绘图应用 —— 无需鼠标和键盘，通过中文语音指令完成创作。

## 快速开始

1. 使用 **Chrome** 或 **Edge** 浏览器
2. 在项目目录启动本地服务器：

```bash
python -m http.server 8080
```

3. 打开 http://localhost:8080
4. 允许麦克风权限
5. 说 **「开始监听」**，然后说出绘图指令

## 常用指令

| 语音 | 效果 |
|------|------|
| 开始监听 / 停止监听 | 控制语音识别 |
| 画一个红色的圆 | 绘制红色圆形 |
| 在中心画蓝色矩形 宽200 高100 | 指定位置与尺寸 |
| 从左上到右下画一条线 | 绘制直线 |
| 画一个红圆然后画蓝色三角形 | 复合指令 |
| 撤销 / 清空画布 | 撤销或清空 |
| 帮助 | 查看指令列表 |
