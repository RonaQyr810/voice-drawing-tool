# AI 语音绘图工具

纯语音控制的 Web 绘图应用 —— 无需鼠标和键盘，通过中文语音指令完成创作。

## 演示视频

📹 **演示录屏**：[assets/demo/demo.mp4](./assets/demo/demo.mp4)

演示内容：启动应用 → 麦克风音量条 → 语音绘图指令 → 撤销操作

## 快速开始

1. 使用 **Chrome** 或 **Edge** 浏览器
2. 双击 **`打开网页.bat`** 或 **`start.bat`** 启动本地服务器
3. 浏览器自动打开 http://127.0.0.1:8080
4. 允许麦克风权限，对着麦克风说话即可绘图

```bash
# 或手动启动
python server.py 8080
```

## 常用指令

| 语音 | 效果 |
|------|------|
| 开始监听 / 停止监听 | 控制语音识别 |
| 画一个红色的圆 | 绘制红色圆形 |
| 在中心画蓝色矩形 宽200 高100 | 指定位置与尺寸 |
| 从左上到右下画一条线 | 绘制直线 |
| 画一个红圆然后画蓝色三角形 | 复合指令 |
| 撤销 / 清空画布 | 撤销或清空 |
| 帮助 | 查看指令列表 |
